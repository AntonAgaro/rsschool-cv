Code example

Codewars kata solution

Maximum subarray sum kata from Codewars:

Task consists in finding the maximum sum of a contiguous subsequence in an array of integers.

Solution:


          const maxSequence = arr => {
            let maxSum = 0;
            let particularSum = 0;
            for (let item of arr) {
              particularSum += item;
              maxSum = Math.max(particularSum, maxSum);
              if (particularSum < 0) particularSum = 0;
            }
            return maxSum;
          };
